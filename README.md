# AutoDash CSV

Dashboard portfolio basé sur un fichier CSV.

## Objectif

Ce projet montre comment transformer un fichier CSV en outil web clair pour une petite entreprise.

## Fonctionnalités

- Chargement automatique d’un fichier CSV
- Import manuel d’un autre CSV
- Vue d’ensemble avec KPI
- Graphiques avec Chart.js
- Tableau filtrable
- Analyse clients
- Analyse produits
- Export CSV des données filtrées
- Interface responsive
- Onglets de navigation

## Technologies

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Chart.js
